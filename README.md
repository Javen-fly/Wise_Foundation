# Wise_Foundation
APP 基础开发框架
		Wise_Foundation只是一个指导框架，使用MVP+Rxjava+Dargger2+Retrofit+ButterKnife主流框架搭建，也就是说Wise_Foundation只是一个APP框架搭建思路。

# 版本
	1.0.0
	
# 框架介绍
		1、M层、V层、P层之间使用DI高度解耦
		2、提供强大基类
		3、M层设计合理 数据统一管理
